# Бот расписания ДВЮИ МВД России

Телеграм-бот для группы `Ю 16 ПОНБ 2022`: показывает расписание и присылает задания к семинарским/практическим занятиям по номеру темы из расписания.

## Команды

- `пары сегодня`
- `пары завтра`
- `расписание на неделю`
- `расписание следующая неделя`
- `когда налоговое`
- `задания налоговое 26.06`
- `задания наркотики завтра`
- `задания личность пятница`
- `задания предпринимательское`
- `/tasks налоговое 26.06`
- `вопросы на зачёт предпринимательское`
- `вопросы на зачёт наркотики`
- `вопросы на зачёт личность`
- `/exam наркотики`
- `сколько осталось учиться`
- `обновить расписание`

## Как работает поиск заданий

Бот берет из расписания:

- дисциплину;
- тип занятия: `сем` или `пр`;
- номер темы из `topic_code`.

Потом ищет совпадение в `materials.json`.

Пример: если в расписании стоит `Налоговое право`, `пр`, `topic_code = 8`, бот найдет блок `Налоговое право / пр / 8` и отправит задания по теме 8.

## Вопросы на зачёт

Команда `вопросы на зачёт (название дисциплины)` отправляет полный список вопросов для промежуточной аттестации по дисциплине из блока `exam_questions` в `materials.json`.

Название дисциплины распознаётся по тем же алиасам, что и в заданиях: `предпринимательское`, `наркотики`, `личность` (а также `налоговое`, если вопросы для него будут добавлены позже).

Примеры:

- `вопросы на зачёт предпринимательское`
- `вопросы на зачёт наркотики`
- `вопросы на зачёт личность`
- `/exam наркотики`

Если по дисциплине вопросов в базе пока нет, бот сообщит об этом отдельным сообщением.

## Переменные окружения Render

Обязательные:

- `TELEGRAM_TOKEN` - токен бота из BotFather.
- `SCHEDULE_API_TOKEN` - токен API расписания с сайта ДВЮИ.

Необязательные:

- `GROUP_ID` - по умолчанию `553`.
- `GROUP_NAME` - по умолчанию `Ю 16 ПОНБ 2022`.
- `CACHE_REFRESH_HOURS` - по умолчанию `6`.
- `STUDY_END_DATE` - дата окончания учебы в формате `YYYY-MM-DD`. Если не указать, бот попробует взять последнюю дату из загруженного расписания.

## Файлы

- `bot.py` - основной Telegram-бот.
- `scraper.py` - загрузка расписания через API.
- `materials.py` - поиск заданий по дисциплине, типу занятия и теме.
- `materials.json` - база заданий из планов семинарских и практических занятий.
- `requirements.txt` - зависимости.
- `Procfile` - команда запуска на Render.

## Важно

Токены не должны лежать в коде. Если старый токен бота уже был в GitHub, лучше перевыпустить его через BotFather и добавить новый только в переменные окружения Render.
