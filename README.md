# Бот расписания ДВЮИ МВД России

Телеграм-бот для просмотра расписания группы Ю 16 ПОНБ 2022.

## Быстрый старт на Render.com

1. Создай GitHub репозиторий `schedule-bot`
2. Загрузи все эти файлы туда
3. На Render.com создай Web Service (подключи GitHub репо)
4. Добавь переменную окружения API_TOKEN с твоим токеном
5. Готово!

## Файлы

- bot.py - основной бот.
- scraper.py - парсер расписания через API
- requirements.txt - зависимости

## API Токен

Получи токен из DevTools браузера (F12 → Network → запрос к student?)
